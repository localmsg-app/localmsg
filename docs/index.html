Installer

Une commande, ou les étapes manuelles si vous préférez ne pas exécuter un script à l'aveugle.

Une commande

curl -fsSL https://localmsg-app.github.io/localmsg/install.sh | bash

Sur Ubuntu/Debian, ça installe un paquet .deb via apt. Sur les autres distributions, ça télécharge un AppImage dans ~/.local/bin.

Manuellement

  • Ubuntu / Debian — téléchargez localmsg.deb depuis la page des releases, puis : sudo apt install ./localmsg.deb
  • Autres distributions — téléchargez localmsg.AppImage, rendez-le exécutable (chmod +x localmsg.AppImage) et lancez-le directement.

Avec Homebrew

brew tap localmsg-app/localmsg https://github.com/localmsg-app/localmsg && brew install --cask localmsg

Ou en une commande

curl -fsSL https://localmsg-app.github.io/localmsg/install.sh | bash

L'app n'est pas signée par Apple. Au premier lancement, si macOS refuse de l'ouvrir : clic droit sur l'app → Ouvrir, ou xattr -cr /Applications/localmsg.app.

Une commande (PowerShell)

powershell -c "irm https://localmsg-app.github.io/localmsg/install.ps1 | iex"

Manuellement

  • Téléchargez localmsg-setup.exe depuis la page des releases
  • Double-cliquez et suivez l'installeur

Désinstaller

Aucune trace cachée à traquer — chaque méthode d'installation a sa propre commande de retrait.

Ubuntu / Debian (.deb)

sudo apt remove localmsg

AppImage

Supprimez simplement ~/.local/bin/localmsg.AppImage

macOS (Homebrew)

brew uninstall --cask localmsg
brew untap localmsg-app/localmsg

Windows

Paramètres → Applications, ou lancez unins000.exe dans le dossier d'installation.
